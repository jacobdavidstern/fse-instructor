React Counter App
A simple React application built with Vite and styled using Bootstrap. The app demonstrates React fundamentals including component architecture, state management with useState, event handling, and responsive UI design.

Users can:

- Add counters with a custom name
- Increment or decrement each counter
- Delete counters when no longer needed

The layout is responsive and works across desktop and mobile window sizes.

Setup Instructions
Clone the repository

bash
git clone https://github.com/your-username/react-counter-app.git
cd react-counter-app
Install dependencies

bash
npm install
Run the development server

bash
npm run dev
Open the app in your browser at the URL shown in your terminal (usually http://localhost:5173).

### Features

Add Counters: Create a new counter with a custom name.

Increment/Decrement: Adjust the counter value using buttons.

Delete Counters: Remove counters from the list.

Responsive Design: Bootstrap ensures usability across different screen sizes.

State Management: Uses React’s useState hook for managing counter state.

Component Architecture: Organized into reusable functional components.

### Learning Objectives

This project reinforces:

Functional components and JSX syntax

Passing props between parent and child components

Managing multiple state variables with useState

Handling user events (onClick, onChange)

Rendering dynamic lists with .map() and unique keys

Responsive UI design with Bootstrap
