$ Project Challenge: Challenge: The "Vibe" Player!
Goal: Build a functional, single-page music player that showcases your frontend skills.
1. Pick a Genre: (e.g., Lo-Fi, Heavy Metal, Classical, Synthwave)
2. Pick a Theme: (e.g., Retro 80s, Minimalist Apple Style, Grunge, Cyberpunk)
3. Build a Player: A single web page that can play/pause and cycle through at least 3
tracks

Need Audio? Download royalty-free placeholder tracks here:

Tech Stack Options Choose one: - React (Component-based approach) - HTML/CSS/JS
(Vanilla implementation)
