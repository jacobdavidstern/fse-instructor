# What to Work on After Each Session

After each class session, navigate to the **Session folder** for the current week. Inside this folder, you'll find several types of files to help you continue learning:

## Files You'll Find

### 1. resource.md
This file contains:
- A corresponding Codecademy course for modules covered
- Additional learning resources and materials
- **Note:** The Codecademy courses are highly recommended for extra practice and cover topics we might not have time for in class

### 2. Practice Files (in the modules folder)
Look for these practice materials:
- Use these to reinforce what you learned in class

### 3. Project Rubrics (rubric.md)
For project-focused modules, this file includes:
- Clear rubric for earning project completion
- Suggested project due dates
- How to submit the project (please note projects are not graded but can be reviewed during office hours)
- **Recommendation:** Complete each project before the next one begins to stay on track
