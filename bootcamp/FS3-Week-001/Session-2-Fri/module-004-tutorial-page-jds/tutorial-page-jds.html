Basic HTML Structure

Every HTML document begins with a <!DOCTYPE html> declaration, followed by the root <html> element. Inside it are two key children: <head> for metadata and <body> for visible content.

The browser parses HTML into a tree-based structure called the Document Object Model (DOM) . Each element becomes a node, starting from the <html> root.

Code Example

       
        <!doctype html>
        <html lang="en">
        <head>
            <meta charset="UTF-8">
            <title>Page Title</title>
        </head>
        <body>
            <h1>Main Heading</h1>
            <p>Your content here</p>
        </body>
        </html>

Text Formatting

HTML provides many elements for text formatting, including: strong importance (commonly rendered as bold), stress emphasis (commonly rendered as italics), highlighted text , small print , and inline code (commonly rendered in monospace font).
