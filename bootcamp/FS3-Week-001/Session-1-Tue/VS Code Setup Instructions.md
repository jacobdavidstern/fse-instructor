# Quick Guide

## 1. Download and Install Visual Studio Code

**Download link:** https://code.visualstudio.com/

> **Note:** During installation on Windows, add all of the checkmarks for:
>
> - "Create Desktop icon"
> - "Add Open with Code" (both options)
> - "Register Code as editor"
> - "Add to Path"

## 2. Add the following extensions to VSCode

**How to install Extensions:** https://code.visualstudio.com/docs/configure/extensions/extension-marketplace

Required extensions:

- **Live Preview**
- **W3C Web Validator**
- **Prettier**

## 3. Update the following Settings

**How to get to settings:** https://code.visualstudio.com/docs/configure/settings#_user-settings

Search for these settings in the search bar and configure as shown:

| Setting                           | Value                   |
| --------------------------------- | ----------------------- |
| Autosave                          | AfterDelay              |
| Format On Save                    | ✓ Check                 |
| Format On Paste                   | ✓ Check                 |
| Default Formatter                 | Prettier (About 8 down) |
| webvalidator - Show Popup         | ✗ Uncheck               |
| webvalidator - Validate on Save   | ✓ Check                 |
| webvalidator - Show Info          | ✗ Uncheck               |
| Live Preview: Open Preview Target | External Browser        |
