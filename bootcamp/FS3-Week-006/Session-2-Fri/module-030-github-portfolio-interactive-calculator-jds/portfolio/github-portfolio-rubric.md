# Github Portfolio Starter Project

## Hints:

- [Repo Creation Walkthrough](https://docs.github.com/en/pages/quickstart)

### Must Have (Required for Passing):

- [x] Setup a basic portfolio page
- [x] Clone Locally
- [x] Can access via (username.github.io)
- [x] Upload your existing project portfolio OR use a template (like https://github.com/rammcodes/Dopefolio)
- [x] Add 2-3 projects to the portfolio
- [x] Add About Section

### Could Have (Bonus Points):

- [ ] Add technology badges showing your skills so far (you can update later)
- [ ] Add a parallaxing affect or other cool animation effect to the pace
